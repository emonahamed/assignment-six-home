1.Describe the differences among var, let and const?

var variable is a functional scope variable. It can be declared and accessed without initialization. It can be updated or reinitialized unlike let and const variable. let and const is appeared first in ES6. Both of them are block scope variable. Neither they can be reinitialized nor declared before initialization. The variable declared by Let can be updated but const can not be updated.

2. Describe the differences between arrow function and regular function.

While both regular and arrow functions work in a similar manner, yet there are certain interesting differences between them : 1. Unlike regular functions, arrow functions do not have their own 'this' keyword. 2. Arguments objects are not available in arrow functions, but are available in regular functions. 3. Regular functions are ‘constructible’ and ‘callable’. The arrow functions are only ‘callable’ but not constructible. Regular functions can be called using the ‘new’ keyword. But we will get a run-time error on trying to construct a non-constructible arrow functions using the new keyword.

3. Why should you use 'Templete String' ?

Template string is a powerful feature of modern JavaScript released in ES6. It lets us insert variables and expressions into strings without needing to concatenate like in older versions of JavaScript. It allows us to create strings that are complex and contain dynamic elements.
